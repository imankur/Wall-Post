Wall-Post
=========

####Backbone Application

This is a SPA(Single Page Application) which retrieve records from mysql database and add new record.

####Technology Used:
HTML,    
CSS,  
Java Script,  
Jquery,  
Backbone,  
Underscore,  
PHP,  
Mysql  

####Database Schema :  
Table Name = wall

######Column:  
ID    |  int(2) //primary key, auto
