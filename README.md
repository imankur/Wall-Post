Wall-Post
=========

Backbone Application

This is a SPA(Single Page Application) which retrieve records from mysql database and add new record.

Technology Used:
HTML,
CSS,
Java Script,
Jquery,
Backbone,
Underscore,
PHP,
Mysql

Database Schema : 

Table Name = 'wall'
===================================
| Column  | 	Type 	     |   Null |
===================================
|   ID    | 	int(2)     |  	No 	 | //Primary Key, Auto Increment 
|  name   | varchar(20) |  	No 	 |
| posts   | varchar(50) | 	 No   |
===================================
